# IP 地址规划表 Ping 工具 - 交互式流程说明

## 🎯 新的交互流程（三步式）

### 流程概览

```
第一步：选择环境（项目）
   ↓ 选择或回车
   ↓
第二步：选择 Sheet（必选）
   ↓ 必须选择
   ↓
第三步：颜色过滤
   ↓ 是/否
   ↓
开始 Ping 测试
```

---

## 📋 详细流程

### 第一步：选择环境

```bash
$ ping-ip-planning

欢迎使用 IP 地址规划表 Ping 工具
======================================================================

======================================================================
可用的配置环境:
======================================================================
  1. 金茂
  2. xxidc
  3. xx项目
======================================================================

请选择环境（输入序号，回车进入手动输入）: 
```

**操作选项：**
- 输入 `1` → 选择金茂项目
- 输入 `2` → 选择 xxidc 项目
- 输入 `3` → 选择 xx 项目
- 直接回车 → 进入手动输入模式

### 第二步：选择 Sheet（必选）

```
✓ 已选择环境: 金茂
  文件: pass/IP地址规划表-金茂1.xlsx

======================================================================
选择要测试的 Sheet 页:
======================================================================
  1. network&security（网络和安全）
  2. server&security（服务器和安全）
======================================================================

请选择 Sheet（必选，输入 q 退出）: 
```

**操作选项：**
- 输入 `1` → 测试网络和安全设备
- 输入 `2` → 测试服务器和安全设备
- 输入 `q` → 退出程序

**注意：这一步必须选择，不能跳过！**

### 第三步：颜色过滤

```
✓ 已选择: network&security

======================================================================
颜色过滤设置:
======================================================================
是否只 ping 绿色单元格？(y/n) [n]: 
```

**操作选项：**
- 输入 `y` → 只 ping 绿色单元格
- 输入 `n` 或直接回车 → ping 所有设备

### 开始测试

```
✓ 不过滤颜色，ping 所有设备

============================================================
IP 地址规划表 Ping 工具
============================================================
```


## 🎨 设计理念

### 为什么是三步式？

1. **第一步：环境选择**
   - 目的：快速切换不同项目
   - 设计：简洁明了，只显示项目名称
   - 退出：回车进入手动模式

2. **第二步：Sheet 选择（必选）**
   - 目的：明确测试目标
   - 设计：强制选择，不能跳过
   - 原因：避免误操作，确保用户清楚测试范围

3. **第三步：颜色过滤**
   - 目的：灵活控制测试范围
   - 设计：简单问答，默认不过滤
   - 原因：大多数情况下不需要过滤

### 优点

- ✅ **简单直观**：每一步都很清楚
- ✅ **不易出错**：必选项避免误操作
- ✅ **快速使用**：熟悉后可以快速输入
- ✅ **灵活性**：可以随时退出或手动输入

## 💡 使用技巧

### 快速操作

对于熟练用户，可以快速连续输入：

```bash
$ ping-ip-planning
1↵    # 选择环境
1↵    # 选择 Sheet
↵     # 不过滤（直接回车）
```

### 退出方式

- 第一步：直接回车 → 进入手动输入模式
- 第二步：输入 `q` → 退出
- 任何时候：`Ctrl+C` → 强制退出

### 命令行模式

如果你记得所有参数，可以直接使用命令行模式：

```bash
# 跳过所有交互
ping-ip-planning -f pass/文件.xlsx -s "network&security" -c none
```

## 📝 常见问题

### Q1: 如何添加新项目？

编辑 `configs/ip_planning_profiles.yaml`，添加新的项目配置。

### Q2: 第二步必须选择吗？

是的，这是故意设计的。确保用户明确知道要测试哪个 Sheet。

### Q3: 可以跳过交互吗？

可以，使用命令行参数：
```bash
ping-ip-planning -f file.xlsx -s "network&security"
```

### Q4: 如果文件不存在会怎样？

系统会提示文件不存在，并询问是否重新选择。

### Q5: 颜色过滤找不到 IP 怎么办？

说明表格中没有绿色单元格，选择不过滤颜色即可。
