## 💡 学习路径建议

### 路径 1：零基础用户（1-2 小时）
```
1. 阅读 QUICKSTART.md （10 分钟）
   ↓
2. 按步骤安装 （10 分钟）
   ↓
3. 第一次运行测试 （5 分钟）
   ↓
4. 遇到问题？查看 FOR_BEGINNERS.md 的故障排查 （10 分钟）
   ↓
5. 熟练使用，阅读 README.md 了解更多功能 （30 分钟）
```

### 路径 2：技术用户（30 分钟）
```
1. 阅读 README.md 快速开始 （5 分钟）
   ↓
2. 安装并运行 （5 分钟）
   ↓
3. 阅读 TOOLS_COMPARISON.md 了解工具区别 （10 分钟）
   ↓
4. 探索高级功能（从服务器 ping、环境管理等） （10 分钟）
```

### 路径 3：开发者（1-2 小时）
```
1. 阅读 README.md 和 ARCHITECTURE.md （20 分钟）
   ↓
2. 阅读 DEVELOPMENT.md 设置开发环境 （15 分钟）
   ↓
3. 运行测试 make test （5 分钟）
   ↓
4. 阅读 TESTING.md 了解测试覆盖 （10 分钟）
   ↓
5. 阅读 CONTRIBUTING.md 了解贡献流程 （10 分钟）
```

## 📞 获取帮助

### 文档中找不到答案？

1. **搜索文档**：
   ```bash
   grep -r "关键词" docs/
   ```

2. **查看示例**：
   - [examples/README.md](../examples/README.md)
   - [examples/credentials.example.xlsx](../examples/credentials.example.xlsx)

3. **查看常见问题**：
   - [FOR_BEGINNERS.md](FOR_BEGINNERS.md#常见错误及解决)
   - [README.md](../README.md) 的故障排查部分

4. **提交 Issue**：
   - 描述问题
   - 附上错误信息
   - 说明操作系统和 Python 版本
