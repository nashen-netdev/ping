# 手动输入模式使用指南

当没有预定义环境时，或者选择进入手动输入模式时，系统会引导你完成配置。

---

## 🎯 进入方式

### 方式 1：没有环境时自动进入

```bash
$ ping-ip-planning

欢迎使用 IP 地址规划表 Ping 工具
======================================================================

没有找到任何配置环境

进入手动输入模式...
```

### 方式 2：主动选择进入

```bash
$ ping-ip-planning

可用的配置环境:
  1. 金茂
  2. xxidc

请选择环境（输入序号，回车进入手动输入）: ↵（直接回车）

进入手动输入模式...
```

---

## 📝 交互流程（3 步）

### 第 1 步：输入文件路径

```
======================================================================
手动输入模式
======================================================================
提示: 输入 'q' 或 Ctrl+C 可随时退出

Excel 文件路径 [pass/IP地址规划表-金茂1.xlsx]: 
```

**说明**：
- 可以输入绝对路径或相对路径
- 支持 `~` 符号（自动展开为用户目录）
- 直接回车使用默认值
- 系统会自动检查文件是否存在

**示例**：
```
# 绝对路径
/Users/sen/Desktop/IP地址规划表-金茂1.xlsx

# 相对路径
pass/IP地址规划表-金茂1.xlsx

# 使用 ~ 符号
~/Desktop/IP地址规划表-金茂1.xlsx

# 默认值（直接回车）
[回车] → 使用 pass/IP地址规划表-金茂1.xlsx
```

**文件检查**：
```
✓ 文件存在: /Users/sen/Desktop/IP地址规划表-金茂1.xlsx
```

如果文件不存在：
```
✗ 文件不存在: /path/to/file.xlsx
  重新输入? (y/n) [y]:
```

---

### 第 2 步：选择 Sheet 页

```
======================================================================
选择要测试的 Sheet 页:
======================================================================
  1. network&security（网络和安全）
  2. server&security（服务器和安全）
======================================================================

请选择 Sheet（输入 q 退出）:
```

**说明**：
- 只能选择 1 或 2
- 不能自由输入（避免错误）
- 输入 `q` 退出

**示例**：
```
请选择 Sheet（输入 q 退出）: 1
✓ 已选择: network&security
```

---

### 第 3 步：颜色过滤

```
======================================================================
颜色过滤设置:
======================================================================
是否只 ping 绿色单元格？(y/n) [n]:
```

**说明**：
- `y` = 只 ping 绿色单元格
- `n` 或回车 = ping 所有设备

**示例**：
```
是否只 ping 绿色单元格？(y/n) [n]: y
✓ 只 ping 绿色单元格
```

或

```
是否只 ping 绿色单元格？(y/n) [n]: ↵（回车）
✓ 不过滤颜色，ping 所有设备
```

---

### 自动配置项（无需输入）

以下配置会自动设置，无需用户输入：

#### ✅ 排除删除线
- **始终自动排除**删除线的 IP 地址
- 无需询问

#### ✅ 本地 ping（智能判断）
- **network&security**：自动使用本地 ping
- **server&security**：询问是否使用本地 ping

```
# network&security 自动设置
✓ network&security 使用本地 ping

# server&security 询问
使用本地 ping? (y/n) [n]:
```

#### ✅ 并发数
- 自动根据系统资源和 ping 方式设置
- 无需手动配置

---

### 第 4 步：配置确认

```
======================================================================
配置确认:
======================================================================
  文件: /Users/sen/Desktop/IP地址规划表-金茂1.xlsx
  Sheet: network&security
  颜色过滤: 无
  排除删除线: 是（自动）
  本地 ping: 是
  并发数: 自动
======================================================================

确认以上配置? (y/n) [y]:
```

**说明**：
- 显示所有配置项供确认
- 直接回车或输入 `y` 确认
- 输入 `n` 取消并退出

## 💡 使用技巧

### 1. 快速使用默认值

如果你经常使用默认配置，可以一路回车：

```bash
$ ping-ip-planning
↵  # 进入手动输入
↵  # 使用默认文件
1  # 选择 network&security
↵  # 不过滤颜色
↵  # 确认
```

### 2. 使用相对路径

如果 Excel 文件在项目的 `pass/` 目录：

```bash
Excel 文件路径: pass/IP地址规划表-项目名.xlsx
```

### 3. 使用 ~ 符号

如果文件在用户桌面：

```bash
Excel 文件路径: ~/Desktop/文件.xlsx
```

### 4. 随时退出

在任何输入提示符时输入 `q` 可以退出：

```bash
Excel 文件路径: q
已取消
```

或使用 `Ctrl+C`：

```bash
Excel 文件路径: ^C
已取消
```

## 🔄 与环境模式对比

### 手动输入模式

**优点**：
- ✅ 灵活，可以测试任意文件
- ✅ 无需预先配置
- ✅ 适合临时测试

**缺点**：
- ❌ 每次需要输入
- ❌ 不适合频繁使用

**适用场景**：
- 临时测试新文件
- 一次性测试
- 文件路径经常变化

### 环境模式（推荐）

**优点**：
- ✅ 预定义配置，选择即用
- ✅ 适合频繁使用
- ✅ 避免重复输入

**缺点**：
- ❌ 需要预先配置环境

**适用场景**：
- 日常测试固定项目
- 多个项目需要频繁切换
- 团队共享配置

## 📚 建议

### 何时使用手动输入模式？

1. **临时测试**：文件在桌面，测试后就删除
2. **新项目评估**：还不确定是否需要创建环境
3. **快速验证**：验证某个文件的 IP 连通性

### 何时创建环境？

如果满足以下条件，建议创建环境：

1. 需要频繁测试（每周 > 2 次）
2. 文件路径固定
3. 团队其他成员也需要使用

创建环境非常简单：

```bash
# 使用手动模式测试后，如果满意可以创建环境
ping-env-add myproject /path/to/file.xlsx -n "我的项目"
```

## ❓ 常见问题

### Q1: 文件路径输入错误怎么办？

系统会提示重新输入：

```
✗ 文件不存在: /wrong/path.xlsx
  重新输入? (y/n) [y]: y
```

### Q2: 可以中途退出吗？

可以！任何时候：
- 输入 `q` 退出
- 按 `Ctrl+C` 强制退出

### Q3: 配置确认后还能修改吗？

配置确认时选择 `n` 可以取消：

```
确认以上配置? (y/n) [y]: n
已取消
```

然后重新运行命令。

### Q4: 为什么 network&security 不问本地 ping？

因为网络设备（network&security）通常需要本地网络环境，自动使用本地 ping 更合理。

服务器（server&security）可能需要通过跳板机访问，所以保留选项。

### Q5: 可以保存手动输入的配置吗？

可以！如果觉得配置不错，创建环境即可：

```bash
ping-env-add myenv /path/to/file.xlsx -n "我的环境"
```

下次就可以直接选择这个环境了。
